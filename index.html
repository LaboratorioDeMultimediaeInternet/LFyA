Para realizar la transformación de una gramática a gramática propia, se debe colocar en el reacuadro "Producciones", la lista de producciones de la gramática.

El formato de la gramática debe ser el siguiente:

  1. Para los símbolos No Terminales se emplean letras mayusculas
  2. Para los simbolos terminales se emplean letras minusculas y números
  3. La producción vacía se identifica por: #

Para colocar las reglas de producción de un símbolo No Terminal se debe poner entre comillas dobles el símbolo, seguido de dos puntos y una lista (delimitada por "[]") con las producciones correspondientes al símbolo

Cada produccion se coloca entre comillas dobles y se separa por una coma

Ejemplo de gramatica válida:

"A":["Caba", "Baca", "aba", "#", "aba"],
"B":["c", "Cba", "AB","d"],
"C":["Caca", "Coco","Adios"],
"D":["a", "3a"]
